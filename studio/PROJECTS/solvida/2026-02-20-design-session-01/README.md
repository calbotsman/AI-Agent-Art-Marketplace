# Sol Vida Design Session 01 (2026-02-20)

This is a runnable design session packet based on the `clawd` studio workflow.

## Session flow
1. `00_intake.md` (scope + constraints)
2. `01_strategy.md` (Rowan)
3. `02_creative-direction.md` (Zara)
4. `03_execution-handoffs.md` (Design / Copy / Producer)
5. `04_critique-gate-round-1.md` (Zara review)

## Output paths
- Designer outputs: `./artifacts/designer/`
- Copy outputs: `./artifacts/copy/`
- Producer outputs: `./artifacts/producer/`

## Definition of done (round 1)
- One selected direction with a clear one-line idea sentence.
- A 6-surface series proof (IG post, IG story, landing hero, email header, product label, pitch slide).
- Critique completed with concrete keep/kill/change actions and next-round tests.
