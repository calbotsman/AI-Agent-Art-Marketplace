# Pre-Flight Checklist — Graphic Design

Use this on **every** design deliverable (social, poster, deck slide, landing, etc.).

## Hierarchy
- [ ] In 3 seconds, a stranger can tell what it is and what to do next
- [ ] Only **one** dominant gesture (scale/weight/color/placement)

## Typography
- [ ] <= 2 type families (unless a deliberate system calls for more)
- [ ] Display type tracking adjusted (not default)
- [ ] Body measure is readable (≈45–75 chars/line) or intentionally tighter

## Grid / spacing
- [ ] Consistent margins and alignment; no "almost aligned" edges
- [ ] Spacing uses a defined scale (e.g., 8/16/24/32…)

## Color / contrast
- [ ] Accent color used sparingly; clear role-based palette
- [ ] Text contrast passes in real conditions (don’t trust the canvas)

## Imagery
- [ ] Crop is intentional; image supports the hierarchy

## Information design (if data present)
- [ ] Remove decoration; labels are clear and close to what they label

## Production
- [ ] Export size correct; type is crisp; checked on target device/app
