# LOOP — Graphic Design Standards

## Goal
Turn references + critiques into **enforceable design rules** that Zara (taste) and Designer (execution) both recall.

## Loop (every project / every round)
1) **Ingest**
   - read relevant references (from dossier + prior rules)
   - collect constraints (format, audience, channel, brand posture)
2) **Create**
   - Designer produces 1–3 variants
3) **Critique**
   - Zara (or Josh) gives critique in: *Keep / Kill / Change* + why
4) **Extract rules**
   - convert critique into:
     - **Rule** (general)
     - **Example** (specific)
     - **Test** (how we’ll verify next time)
5) **Update artifacts**
   - Update `STANDARDS.md` (durable) + `CHECKLIST.md` (pre-flight)
6) **Enforce next round**
   - every new output must explicitly reference the checklist items it satisfies

## Storage conventions
- New rules go into `STANDARDS.md` under the closest section.
- Project-specific rules also get copied into that project’s own LOOP doc (if present).
