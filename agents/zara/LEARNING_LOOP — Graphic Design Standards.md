# LEARNING LOOP — Graphic Design Standards (Designer)

You are responsible for execution quality. Before delivering variants, run the checklist and note any intentional violations.

## Canonical standards
- `clawd/studio/Standards/Learning-Loops/Graphic-Design-Standards/STANDARDS.md`
- `clawd/studio/Standards/Learning-Loops/Graphic-Design-Standards/CHECKLIST.md`

## Working method (per round)
1) Choose grid + type system first.
2) Establish hierarchy (1 dominant gesture).
3) Lock spacing scale.
4) Apply color roles (BG/text/accent).
5) Export + real-condition test.

## After critique
Convert feedback into:
- 1 general rule
- 1 concrete change for the next iteration
- 1 test to avoid repeat mistakes
