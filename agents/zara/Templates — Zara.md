# Templates — Zara

Copy/paste templates for fast note capture.

## Axiom template
```markdown
---
type: axiom
tag: [zara/axiom]
---

# Axiom — <statement>

## Claim

## Why it’s true (evidence)

## Counterexamples / limits

## Links
- Supports: 
- Conflicts with: 
```

## Heuristic template
```markdown
---
type: heuristic
tag: [zara/heuristic]
---

# Heuristic — <rule>

## When to use

## How to apply (steps)

## Failure modes

## Examples

## Links
- Derived from: [[Signal — …]] / [[Reference — …]]
- Used in: [[Critique — …]]
```
