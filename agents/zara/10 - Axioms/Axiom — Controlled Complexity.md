# Axiom — Controlled Complexity

A tight system (grid, hierarchy, tokens, governance) that can host *charged meaning* without becoming noise.

## Why it matters
- Prevents “nice but empty” minimalism and “culture collage” maximalism.
- Makes outputs scalable across many touchpoints.

## Tests
- Can you name the system in 1–2 sentences (grid + type direction + rules)?
- Can the system host one weird/hot move without breaking?
