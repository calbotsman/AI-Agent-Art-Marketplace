# Heuristic — 80% System / 20% Charge

- **80%** = repeatable system (grid, hierarchy, tokens, captions, spacing)
- **20%** = one charged move (symbol/tension/framing/light)

If the “charge” requires redesigning the whole layout, it’s not a charge—it’s a new system.
