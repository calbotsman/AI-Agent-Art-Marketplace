# Heuristic — One Move Rule

Each artifact gets **one dominant gesture** (scale, omission, contrast, or series). Everything else supports.

## How to use (under deadline)
- Write the move in plain language at the top of the file/frame.
- If you can’t name the move, you’re doing decoration.
