# Zara — Voice + Copy System (derived from the Influence Library)

A working codebook for writing that feels **systemic + charged**: high signal, calm surface, sharp point of view.

Built from the library’s core tensions:
- **Rigor / integrity** (Bense, Tufte, Rams, Vignelli, NYT)
- **Meaning / culture / ethics** (Sontag, Jung, Bürger, 032c, Purple)
- **Restraint with a single dominant gesture** (Hara, Fukasawa, Mies)
- **World + codes** (Ralph, YSL)
- **Citation / remix / distribution** (Virgil, Duchamp)

---

## 1) Voice pillars (the non‑negotiables)

### Pillar A — High signal, low noise
Write like every word costs money.
- Prefer **specific claims** over adjectives.
- If a sentence can be shorter without losing meaning, cut it.
- Make hierarchy with spacing, punctuation, and structure—not hype.

### Pillar B — Calm surface, sharp edge
The tone is composed; the ideas are not soft.
- Understatement carries authority.
- One clean line can hold menace, humor, or desire.

### Pillar C — Systems, not slogans
Zara doesn’t “vibe.” Zara **operates**.
- Use rules, principles, constraints, checklists.
- Show how the system scales (series thinking > one-off poetry).

### Pillar D — Editorial, not promotional
Write like a magazine with taste.
- Observation-first.
- A stance, not a pitch.
- Let the reader feel respected (no begging, no over-explaining).

### Pillar E — Evidence + integrity
No aesthetic lying.
- Put context next to claims (numbers, baselines, sources, definitions).
- Never use moral language to cover weak work.

### Pillar F — Charged meaning (symbol, ritual, myth)
The system has **codes**.
- Repeating motifs, names, archetypes.
- Worldbuilding through consistent objects, behaviors, and language.

### Pillar G — One dominant gesture
Each piece gets **one** strong move, then stops.
- A single provocation.
- A single contrast.
- A single sentence that rearranges the room.

## 2) Banned phrases + banned cadences

### Banned phrases (anti-Sontag / anti-corporate)
Avoid anything that sounds like a deck template:
- “Game-changer” / “disrupt” / “revolutionary”
- “Elevate” / “level up” / “next level”
- “Best-in-class” / “world-class” / “state-of-the-art”
- “Seamless” (unless you specify what became seamless)
- “Unlock” / “unleash” / “supercharge”
- “Journey” / “story” (unless it’s literal)
- “Curated experiences” / “bespoke solutions”
- “Authentic” / “premium” / “timeless” (unless you prove it)
- “We’re passionate about…”
- “At the end of the day…”
- “In today’s fast-paced world…”
- “Designed to” + vague verb (“designed to inspire,” “designed to empower”)

### Banned cadences (anti-hype rhythm)
- **TED crescendo**: short lines that only get louder and emptier.
- **Purpose fog**: big moral nouns with no operational detail.
- **Hashtag pep talk**: “You’ve got this” energy.
- **Luxury perfume vagueness**: endless sensual adjectives, zero substance.
- **Over-wink quoting**: quotation marks as a crutch (Virgil method without the thesis).

Replacement rule: if you feel tempted to hype, switch to **a constraint, a number, a proof, or a crisp refusal**.

## 4) 10 template-less copy moves (repeatable techniques)

1. **The Idea Sentence** (Gate 1)
   - State the concept in one line a non-designer can repeat.
   - Then stop.

2. **Constraint-as-credibility**
   - Lead with a limitation or rule. Make it feel chosen.
   - “Two colors. One typeface. Zero excuses.”

3. **Annotation captions** (Tufte / NYT)
   - Use footnote energy: short clarifications that add context, not fluff.
   - Make the reader smarter in 8 words.

4. **Series logic** (Albers / Saville)
   - Talk in sets, iterations, variants, controlled deltas.
   - “Same system. New pressure.”

5. **The single dominant gesture** (Hara / Fukasawa)
   - One strong verb or image; everything else recedes.
   - If you add a second gesture, remove the first.

6. **Framing > fabrication** (Duchamp)
   - Change meaning by naming, sequencing, or context.
   - “It’s not a collection. It’s a standard.”

7. **Editorial provocation with receipts** (Kalman / Sontag)
   - Say the blunt thing, then cite the reason.
   - Heat + accountability.

8. **Myth via codes** (Ralph / Jung)
   - Repeat a few symbols/objects/rituals until they become identity.
   - The reader should recognize a line at distance.

9. **Black/white authority** (YSL)
   - Use stark contrast and declarative sentences.
   - No pleading, no filler, no “maybe.”

10. **Distribution-aware language** (Virgil)
   - Write with shareability and modularity in mind.
   - Fragments that travel without losing meaning.
