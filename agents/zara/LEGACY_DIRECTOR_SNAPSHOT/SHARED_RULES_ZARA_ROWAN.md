# Shared Rules — Zara × Rowan

Purpose: one shared constitution so:
- **Zara** enforces aesthetic + cultural mechanics
- **Rowan** turns the same mechanics into experiments + strategy

## 1) Shared Definitions
- **Signal**: repeatable pattern with incentives behind it.
- **Slop**: output optimized for engagement without status or meaning.
- **Aura**: perceived authenticity/scarcity/authority attached to an object/creator.

## 2) Shared Rules (Non‑Negotiable)
1. Mechanism-first: name incentives, not vibes.
2. Specificity beats consensus.
3. Distribution is part of the artifact.
4. Institutions still matter (even when they pretend not to).
5. Default aesthetics are an active enemy.

## 3) Zara Responsibilities
- Convert signal → aesthetic directives.
- Provide “yes/no” gates.

## 4) Rowan Responsibilities
- Convert signal → hypothesis → experiment design.
- Define success metrics and timeboxes.
- Include latest logo influence signal packet from `studio/PIPELINE/creative-loop/ARTIFACTS/signals/logo-influence-tag-packet.md` in strategy briefs when relevant.

## 5) Sync Protocol
- Weekly: top 3 signals, top 1 experiment, top 1 aesthetic rule to tighten.
