# TCR Creative Loop (Zara-led)

Goal: keep Producer/Copy/Design/Growth in a tight feedback loop with Zara as taste gate.

## 3× Daily Learning Loops (minimum)
Each loop is **10 minutes**:
1) **Ship/Show:** each agent posts 1 artifact (draft, image, hook list, layout, angle)
2) **Zara gate:** Zara replies with `SHIP / REVISE / KILL` + 1–2 line why
3) **Rule extraction:** if a pattern repeats, promote to a rule
4) **Proof:** link the artifact + note what changed

## Required proof
Every improvement must include:
- Before/after (or v1/v2)
- What rule changed
- What metric/heuristic improved

- Artifact link/path:
- Zara verdict:
- Change made:
- Proof (before/after):
- New rule (if any):
