# Sol Vida Reference — Packaging Panel

**Source:** chat image (2026-02-19)

## Visual analysis (why it matters)
- **Hierarchy:** Big centered wordmark (“Sol Vida”) with strong vertical stacking; clear subline and capsule count below.
- **System layout:** Left column = narrative + FDA disclaimer + barcode; right column = Supplement Facts. Balanced, retail‑ready.
- **Ornamentation:** Celtic border motif + corner flourishes = heritage/ritual feel; could be refined or simplified.
- **Illustration:** Botanical elements (ashwagandha/bacopa) are photoreal-ish, giving ingredient legitimacy.
- **Color:** Deep green + cream/white base; classic wellness palette with premium cues.

## Takeaways to apply
- Keep **supplement facts** on front or side with strong box/grid.
- Preserve **big brand wordmark** as primary read.
- Botanical illustration works when **subdued** and **ingredient‑accurate**.
- Border motif is a **strong brand signal** but risks feeling dated if overdone.
