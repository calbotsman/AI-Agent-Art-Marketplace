# Gregory Crewdson

## 5-sentence summary
Crewdson makes photographs like feature films: massive crews, constructed sets, and lighting that feels like a suspended narrative. The images sit in uncanny suburban Americana—beautiful, eerie, emotionally stalled—where something happened or is about to. His work is a masterclass in *production value serving story*: every lamp, fog bank, and color temperature is intentional. The brand lesson is that a single image can carry a whole world if the narrative tension is precise. For Zara, Crewdson is the bar for cinematic stills: controlled light + psychological subtext.

## Steal this (8)
1. Light like cinema (motivated sources, layered practicals).
2. Build narrative tension: unanswered questions beat explicit story.
3. Use production design as character.
4. Control color temperature for mood.
5. Compose with depth and negative space to create unease.
6. Make “ordinary” locations feel mythic.
7. Commit to pre-production (storyboards, scouts) like film.
8. Keep faces restrained; emotion lives in environment.

## Concrete modern applications (3)
- **Luxury fashion editorials**: cinematic stills with narrative ambiguity.
- **Tech brand films**: still frames designed to stand alone as key art.
- **Hospitality campaigns**: place-as-psychology storytelling.
