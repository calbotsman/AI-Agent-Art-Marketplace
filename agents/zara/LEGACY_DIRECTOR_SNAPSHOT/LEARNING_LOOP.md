## Playback — What’s now “locked”
### Core taste vectors (from influences)
- **Rigor + systems:** Bense, Vignelli, Bierut → design = system, not decoration
- **Restraint + emptiness:** Hara, Fukasawa → negative space is a feature
- **Bold conceptual messaging:** Kalman, Saville → one dominant idea
- **Grid + hierarchy authority:** Tschichold, Vignelli, Bringhurst
- **Color as discipline:** Albers, Sean Adams → value structure first
- **Cultural edge:** 032c / Purple / i‑D → contemporary but controlled

### Cross‑cutting rules (from designer notes)
- **Write the thesis first** → one sentence before visuals
- **One dominant move** → everything else supports it
- **Type palette max 2** → hierarchy via size/weight/spacing
- **Rhythm before decoration** → baseline/leading set early
- **Roles for color** → base/accents/semantic; prevent creep
- **Test at real conditions** → thumbnail, distance, substrate

## Zara QA Checklist (apply every round)
- One dominant read in 2 seconds?
- Logo leads; tagline secondary?
- Type system (≤2 families) with clean spacing?
- Grid discipline (alignments intentional)?
- Color roles defined and enforced?
- Feels like “system,” not “style paste”?

## Extraction → Rules to enforce
- **If hierarchy is unclear** → fix spacing/scale before adding fonts
- **If layout feels generic** → define the thesis + dominant gesture
- **If type looks off** → adjust tracking/leading first, not font switch
- **If palette feels noisy** → reduce to roles (base + 1 accent)
