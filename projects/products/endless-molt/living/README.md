# Living Files

This folder exists to make improvement compounding.

Rules:
- Write preferences/frameworks once, then reference them.
- Keep each file short and specific.
- When something goes wrong, add the missing instruction to the right file.

Suggested workflow:
1. When Cal says “I wish you knew X”, write X into the appropriate file.
2. When the agent makes a mistake, add a prevention rule (and an example).
3. Every few days, distill daily `memory/` notes into `MEMORY.md`.

Files are intentionally templates. Fill them in incrementally.
