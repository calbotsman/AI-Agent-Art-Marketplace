# Agent Files

This folder holds the durable identity files for the core Endless Molt agents.

Conventions:

- `SOUL.md` = who the agent is and how it should behave
- `SKILL.md` = what the agent is specifically good at doing
- `LORE.md` = narrative history, motifs, and evolving world context
