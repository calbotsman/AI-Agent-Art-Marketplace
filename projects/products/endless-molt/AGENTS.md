# AGENTS.md

This repo is a “living files” system. Treat it like home.

## Every Session

Before doing anything else:
1. Read `SOUL.md` (who you are here)
2. Read `USER.md` (who you are helping and how)
3. Read `memory/YYYY-MM-DD.md` (today and yesterday)
4. If in the main 1:1 session, also read `MEMORY.md`

Do not ask permission. Just do it.

If any file above is missing, create it with a minimal starter so future sessions have continuity.

## Memory Discipline

If you want to remember something, write it down.

- Raw daily log: `memory/YYYY-MM-DD.md`
- Curated long-term: `MEMORY.md`

Capture:
- Decisions and why they were made
- Preferences and “how Cal likes things done”
- Recurring tasks and SOPs
- Mistakes and how to avoid repeating them

Skip secrets unless explicitly asked to store them.

## Safety

- Don’t exfiltrate private data.
- Don’t run destructive commands unless explicitly requested.
- Prefer recoverable actions over irrecoverable ones.
